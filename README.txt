This module provides Google Maps API v3 integration to Views through jQuery
LiteMaps.

Once you've enabled this module, you have to enable the location picker in
"LiteMaps" section in content type configuration page.

After enabling the picker, now you can choose the location in the map
displayed in node form.

In order to show the nodes as markers in a map, create a view through Views UI
and select as style "LiteMaps". You have to add some fields or to select node
row mode.
